# 12개월 구축 로드맵

## Phase 0: Discovery & Design (0~1개월)

- 고객 세그먼트 2개 선정
- 대표 장비/작업 절차 3개 선정
- 위험도와 규제 요구사항 정의
- 디바이스 후보 테스트
- SOP/PDF/정비 로그 샘플 수집
- 파일럿 KPI 합의

## Phase 1: Professional Pilot Core (2~4개월)

- XR Client 기본 작업 단계 UI
- 음성 질의응답
- PDF/SOP RAG
- 작업 세션/로그/evidence 저장
- 기본 PPE/checklist
- 관리자 콘솔 v0
- 엣지 게이트웨이 v0

## Phase 2: Context & Safety Intelligence (5~7개월)

- 객체/라벨/OCR 인식
- SOP graph/state machine
- safety policy engine
- LOTO/PPE/순서 위반 경고
- 전문가 에스컬레이션
- RAG 평가/레드팀 체계
- 파일럿 현장 shadow mode

## Phase 3: Enterprise Hardening (8~10개월)

- SSO/RBAC/ABAC
- 문서 승인 workflow
- 모델 레지스트리/배포 링
- SIEM/export/audit log
- offline/edge failover
- KPI/ROI dashboard
- 개인정보 redaction 강화

## Phase 4: Commercial Launch (11~12개월)

- 산업별 도메인팩 1~2개 상품화
- 가격표/계약서/SLA/보안 문서 정리
- 고객 성공 playbook
- 파트너 SI/하드웨어 번들 구성
- reference case 확보
- GA 릴리스
