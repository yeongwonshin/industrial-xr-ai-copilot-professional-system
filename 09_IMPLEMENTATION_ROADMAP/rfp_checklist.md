# 고객 RFP/제안서 체크리스트

## 고객에게 확인할 질문

### 현장/작업
- 대상 장비와 작업 절차는 무엇인가?
- 작업 위험도와 필수 PPE는 무엇인가?
- 기존 매뉴얼/SOP의 최신성은 어떤가?
- 작업 중 네트워크 환경은 어떤가?
- 장갑/소음/방진복 등 착용 제약은 있는가?

### 데이터/시스템
- CMMS/MES/ERP/SCADA 연동이 필요한가?
- 원본 영상/음성 저장이 허용되는가?
- 온프레미스/엣지 요구사항은 무엇인가?
- SSO/MDM/SIEM 연동이 필요한가?
- 데이터 보존 기간은 얼마인가?

### 안전/품질
- LOTO/위험성 평가/JSA 문서가 있는가?
- 고위험 작업 승인자는 누구인가?
- 품질 evidence로 인정되는 자료 형식은 무엇인가?
- 감사 대응 시 필요한 리포트는 무엇인가?

## 제안서 포함 항목

- 시스템 개요
- 대상 use case
- 디바이스/배포 구조
- 데이터 처리 방식
- 안전 기능 범위와 한계
- 파일럿 KPI
- 일정/역할/비용
- 보안/개인정보 조치
- 성공 후 확장 로드맵
