# 투자자용 1페이지 요약

## 회사/제품명
**FieldSense XR AI Work Copilot**

## 문제
산업 현장은 숙련자 부족, 높은 교육비, 안전사고, 품질 불량, 복잡한 매뉴얼, 정비 지연에 동시에 노출되어 있습니다. 기존 솔루션은 AR 원격지원, 매뉴얼 뷰어, 작업 체크리스트, MES/CMMS가 분절되어 있어 “작업자의 눈앞 상황”을 기준으로 실시간 판단하지 못합니다.

## 솔루션
스마트글래스/MR 헤드셋에서 AI가 작업자의 시야·음성·손동작·장비 상태·매뉴얼을 통합 분석하여 다음 작업을 안내하고, 위험한 순서나 품질 오류를 즉시 경고하는 XR 기반 작업 코파일럿입니다.

## 왜 지금인가
- 공간컴퓨팅/패스스루 MR 기기 성능 향상
- 기업용 몰입형 훈련·guided work 수요 증가
- LLM/RAG 기반 문서 질의응답과 멀티모달 모델의 실용화
- 숙련자 은퇴와 현장 인력난
- 안전·품질 감사 증빙 자동화 필요성 증가

## 제품 구조
1. XR Client: Unity/OpenXR, 음성/제스처/작업 단계 UI
2. AI Core: 멀티모달 인식, RAG, 정책 엔진, 경고 생성
3. Knowledge OS: 매뉴얼/SOP/정비 로그/도면 수집 및 버전관리
4. Safety Layer: PPE, LOTO, 위험 순서, 품질 검증
5. Enterprise Console: 현장 배포, 로그, KPI, 보안, 권한

## 초기 고객 세그먼트
| 세그먼트 | 높은 지불 이유 |
|---|---|
| 반도체/배터리/자동차 공장 | 불량·다운타임·안전사고 비용이 큼 |
| 병원 의공/장비팀 | 의료장비 정비·검교정 오류 리스크 큼 |
| 항공/철도 정비 | 절차 준수와 감사 증빙 필수 |
| 실험실/화학 플랜트 | PPE/위험물/순서 오류가 사고로 직결 |

## 비즈니스 모델
- Site License: 현장당 월 구독
- Seat License: 작업자/관리자/전문가 계정당 월 과금
- Device Bundle: 헤드셋 세팅·MDM·현장 네트워크 구성비
- Knowledge Pack: 매뉴얼 정제, SOP 그래프화, 장비별 AI 팩
- Premium: 온프레미스, 전용 모델, 규제 산업 감사 리포트

## 핵심 KPI
- 신규 작업자 독립 수행 시간 30~50% 단축
- 재작업/불량률 10~30% 감소
- 원격 전문가 호출 전 자체 해결률 20~40% 상승
- 작업 로그 자동화율 80% 이상
- 위험 행동 경고 precision 90% 이상, recall 80% 이상을 파일럿 목표로 설정

## 방어력
- 산업별 절차 데이터와 작업 로그 축적
- 고객 장비별 지식 그래프
- 안전 정책과 감사 템플릿
- 현장 배포/엣지/보안 운영 노하우
- 모델 성능보다 “현장 통합”이 진입장벽
