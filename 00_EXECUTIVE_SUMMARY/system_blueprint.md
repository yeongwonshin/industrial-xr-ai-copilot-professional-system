# 시스템 블루프린트

## 1. 제품 비전

FieldSense XR AI Work Copilot는 숙련자 부족, 작업자 교육 비용, 안전사고, 품질 불량, 정비 지연을 줄이기 위한 **산업용 XR AI 작업 코파일럿 플랫폼**입니다. 기존 AR 원격지원이나 단순 매뉴얼 뷰어를 넘어, 작업자의 현재 시야·음성·손동작·장비 상태·절차 문서를 결합하여 “지금 무엇을 해야 하는지”와 “하면 안 되는 행동”을 동시에 판단합니다.

## 2. 해결하는 문제

| 문제 | 기존 방식 | FieldSense 방식 |
|---|---|---|
| 신입/비숙련 작업자의 절차 오류 | 종이 매뉴얼, 옆자리 숙련자 의존 | 단계별 XR 안내 + 시야 기반 검증 |
| 위험 작업 순서 위반 | 사후 사고 보고 | PPE/LOTO/금지 순서 실시간 경고 |
| 정비 지식 분산 | PDF, 구두 노하우, 이메일 | 장비별 지식 그래프 + RAG |
| 품질 증빙 부족 | 수기 체크리스트 | 사진/음성/센서 기반 작업 로그 |
| 원격 전문가 부족 | 전화/화상 통화 | 현장 시야 공유 + AI 예비 진단 + 전문가 에스컬레이션 |

## 3. 전문 시스템 범위

### A. XR Workbench
- Unity/OpenXR 기반 MR UI
- 작업 단계 카드, 3D 앵커, 위험 영역 하이라이트
- 음성 질의응답, 시선/손 제스처, 핸즈프리 확인
- 오프라인 캐시와 현장 네트워크 불안정 대응

### B. Context AI Engine
- 객체/부품/계기판/에러코드/PPE 인식
- 손동작·작업 상태 추론
- 문서 RAG + 작업 절차 그래프
- 정책 기반 안전 판단과 근거 제시

### C. Industrial Knowledge OS
- PDF, SOP, CAD/BOM, CMMS, ERP, 과거 정비 로그 수집
- 문서 버전관리, 승인 워크플로우, 장비별 지식팩
- 절차 단위 chunking, 위험 태그, 요구 도구/부품 태그

### D. Safety & Quality Layer
- LOTO, PPE, 2인 승인, 토크값, 순서 의존성, 금지 조건
- 품질 증거 캡처와 감사 로그
- 안전 경고는 AI 단독 판정이 아니라 정책 룰 + 신뢰도 + 사람 승인으로 처리

### E. Enterprise Control Plane
- 테넌트/현장/장비/작업자/역할 관리
- 감사 로그, 대시보드, 파일럿 KPI, SLA 모니터링
- 보안 정책, 데이터 보존 정책, 현장별 모델/문서 배포

## 4. 차별화 포인트

1. **문서 Q&A가 아니라 절차 실행 시스템**: 답변만 제공하지 않고 작업 단계 완료 여부를 검증합니다.
2. **안전 우선 설계**: 위험한 작업은 “추천”이 아니라 “중지·확인·승인” 흐름으로 처리합니다.
3. **멀티모달 근거 결합**: 시야, 음성, 문서, 장비 센서, 작업 로그를 함께 판단합니다.
4. **도메인팩 판매 가능**: 반도체 장비, 병원 장비, 발전소, 항공 정비 등 산업별 확장이 가능합니다.
5. **현장 적용성**: 오프라인/엣지 모드, 보안 구역, 장갑 착용, 소음 환경을 기본 전제로 설계합니다.

## 5. 상용화 가설

- 초기 고객: 제조 공장, 설비 정비업체, 병원 의공팀, 항공/철도 정비, 실험실 안전관리팀
- 구매 부서: 생산기술, EHS, 품질, 교육, 정비, 디지털전환 조직
- 지불 명분: 사고 예방, 교육기간 단축, 재작업 감소, 숙련자 원격지원, 감사 대응
- 과금: 현장당 기본료 + 헤드셋/사용자 구독 + 매뉴얼 구축비 + 도메인팩 + 온프레미스/엣지 옵션
