# 샘플 SOP: Compressor-A7 필터 교체 전 LOTO/밸브 차단

## 목적
Compressor-A7 필터 교체 전 저장 에너지 방출과 우발적 기동을 방지한다.

## 필수 PPE
- 절연 장갑
- 보안경 또는 안면보호구
- 안전화

## 절차

1. 작업 오더 WO 번호와 장비 ID를 확인한다.
2. Compressor-A7 주변 접근 금지 표지를 설치한다.
3. Pump P-2 전원을 OFF로 전환한다.
4. 차단기에 개인 잠금장치와 태그를 부착한다.
5. 밸브 V-302를 CLOSE 위치로 전환한다.
6. 압력계 G-12가 0 bar인지 확인한다.
7. 압력이 남아 있으면 배출 절차를 수행하고 supervisor에게 보고한다.
8. 필터 하우징 분해를 시작한다.

## 고위험 차단 조건

- 전원 OFF 미확인
- LOTO 태그 미확인
- 압력계 0 bar 미확인
- 필수 PPE 미착용

## Evidence

- 장비 ID 이미지
- 차단기 OFF 상태 이미지
- LOTO 태그 이미지
- V-302 CLOSE 상태 이미지
- 압력계 G-12 0 bar 이미지/OCR
