# 문제 정의와 타깃 사용자

## 핵심 문제

산업 현장의 작업자는 다음 네 가지를 동시에 처리해야 합니다.

1. 복잡한 절차를 기억하거나 매뉴얼에서 찾아야 한다.
2. 눈앞의 장비/부품 상태가 절차 조건과 맞는지 판단해야 한다.
3. PPE, LOTO, 압력/온도/전기 등 위험 조건을 지켜야 한다.
4. 작업 완료 증거를 품질/안전 감사에 남겨야 한다.

이 네 가지는 모두 사람이 놓치기 쉬우며, 특히 비숙련자·교대 근무자·다품종 장비 환경에서는 오류가 증가합니다.

## 페르소나

### 1. 신규 현장 작업자
- 니즈: 다음에 무엇을 해야 하는지, 내가 제대로 했는지 즉시 확인
- Pain: 매뉴얼이 너무 길고 작업 중 손으로 찾기 어렵다
- 성공 경험: “지금 밸브 V-302를 잠그고, 압력계가 0 bar인지 확인하세요”라는 시야 기반 안내

### 2. 숙련 정비사
- 니즈: 반복 질문 감소, 복잡한 진단 보조, 기록 자동화
- Pain: 후배 지원과 실제 정비를 병행해야 함
- 성공 경험: 에러코드와 과거 정비 이력을 AI가 요약하고 후보 원인 제시

### 3. EHS/안전 관리자
- 니즈: 위험 작업 순서 준수와 사고 전 예방
- Pain: 현장 순찰과 사후 보고만으로는 한계
- 성공 경험: PPE 미착용, LOTO 누락, 금지 순서가 실시간으로 경고되고 로그가 남음

### 4. 품질 관리자
- 니즈: 불량/재작업 감소, 증거 기반 품질 추적
- Pain: 수기 체크리스트의 신뢰도 부족
- 성공 경험: 부품 방향, 체결 순서, 토크값, 라벨을 자동 확인하고 증거를 저장

### 5. 현장 관리자/공장장
- 니즈: 교육비·다운타임·사고비용 절감
- Pain: 현장별 작업 표준화가 어렵고 숙련자 의존도가 높음
- 성공 경험: 파일럿 KPI 대시보드에서 교육기간, 오류율, 대응시간 개선 확인

## 구매자와 의사결정자

| 역할 | 관심 지표 | 메시지 |
|---|---|---|
| 생산기술 | tact time, 재작업, 장비 가동률 | 절차 오류와 다운타임을 줄입니다 |
| EHS | near miss, 사고, PPE 준수 | 사고가 나기 전에 위험 순서를 차단합니다 |
| 품질 | defect, audit finding, traceability | 증거 기반 품질 로그를 자동 생성합니다 |
| 교육 | ramp-up time, 숙련도 | 신입 교육을 현장 코칭으로 전환합니다 |
| IT/보안 | 데이터 통제, MDM, 네트워크 | 온프레미스/엣지/권한/감사 체계를 제공합니다 |
