# 가치 제안과 수익 모델

## 핵심 가치 제안

### 안전
- PPE 착용, LOTO, 금지 구역, 위험 순서를 실시간 검증
- 위험 행동이 감지되면 답변이 아니라 중지/확인/상급자 승인 플로우로 전환
- 사고 보고가 아니라 사고 전 차단을 목표로 함

### 품질
- 부품 방향, 결합 상태, 라벨/시리얼, 계기값, 절차 완료 증거 저장
- 작업자별·장비별 편차 분석
- 감사 대응용 증적 자동화

### 생산성
- 매뉴얼 검색 시간 감소
- 현장 원격 전문가 호출 횟수 감소
- 신규 작업자 독립 수행 시간 단축
- 반복 교육 콘텐츠를 XR 절차로 재사용

### 지식 보존
- 숙련자 노하우를 절차 그래프와 예외 처리 노드로 구조화
- 정비 이력, 에러코드, 부품 교체 주기를 장비 지식팩으로 축적

## 제품 패키징

| 패키지 | 대상 | 포함 기능 |
|---|---|---|
| Starter Pilot | 1개 라인/장비군 | PDF RAG, 음성 Q&A, 체크리스트, 기본 로그 |
| Professional Site | 공장/병원/정비센터 | 객체 인식, 위험 행동 감지, 관리자 콘솔, KPI 대시보드 |
| Enterprise Control | 다수 현장 | 멀티테넌트, 온프레미스/엣지, SSO, SIEM 연동, 감사 리포트 |
| Regulated Industry | 항공/의료/화학 | 검증 문서, 안전 사례, 변경관리, 강화된 증적 관리 |

## 과금 구조 예시

- 초기 구축비: 3천만~2억 원/현장
- 헤드셋 세팅비: 기기당 50만~300만 원
- 구독료: 현장당 월 300만~2천만 원
- 사용자 과금: 작업자 월 5만~30만 원, 관리자/전문가 월 20만~80만 원
- 지식팩 구축: 장비군당 1천만~1억 원
- 온프레미스/전용 모델: 별도 견적

## ROI 산정 프레임

```text
월간 절감액 =
  재작업 감소액
+ 다운타임 감소액

+ 원격지원/출장 감소액
+ 사고/near-miss 리스크 절감액
+ 감사 대응 시간 절감액
```
