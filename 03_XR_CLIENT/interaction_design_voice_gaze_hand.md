# 음성·시선·손동작 상호작용 설계

## 입력 채널

### 음성
- wake phrase: “필드센스” 또는 고객사 지정어
- noise suppression과 현장용 커스텀 vocabulary 필요
- 중요한 확인은 “예/아니오” 대신 구체적 phrase 사용: “V-302 잠금 확인”

### 시선
- Vision Pro류 기기는 개인정보/플랫폼 제한으로 raw gaze 접근이 제한될 수 있으므로 UI focus/selection abstraction만 사용한다.
- continuous gaze tracking을 안전 판정의 필수 근거로 두지 않는다.

### 손동작
- 핀치, 탭, 스와이프, 체크 gesture
- 장갑/기름/오염 환경에서 인식 실패 가능성을 고려해 음성 대체 제공
- 손동작은 “작업 완료 evidence”가 아니라 보조 신호로만 사용

### 물리 입력
- 산업용 clicker, 링 컨트롤러, 장비 버튼, 풋 스위치 옵션
- 고소음/고위험 작업장에서는 물리 확인 입력을 권장

## 대화 설계

```text
Worker: 이 에러코드 E47 원인이 뭐야?
AI: 장비 A7의 E47은 냉각수 유량 저하와 관련됩니다.
    현재 SOP-MAINT-12 기준으로 먼저 펌프 P-2 전원을 차단하고,
    LOTO 확인 후 필터 F-3 막힘을 점검하세요.
    지금 전원 차단이 확인되지 않아 점검 단계로 진행할 수 없습니다.
```

## 확인 설계

- 저위험 확인: 작업자 음성 확인 + 로그
- 중위험 확인: 시야 evidence + 작업자 확인
- 고위험 확인: 시야 evidence + 정책 조건 + supervisor 승인
- 치명 위험 확인: 시스템 안내 중지 + 현장 안전 절차 안내 + 사람 판단 요구
